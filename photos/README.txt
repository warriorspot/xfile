XFile sample database - photo album

A simple photo album

Examples of how to use it:

- To see photos for a specific event
1. Switch to Data Entry mode
2. Mark -> Mark Records
3. Choose field "event"
4. Check "match parital words" and "ignore case" if you want
5. Enter the event name to match (i.e. "jerusalem" or "washington"
6. Mark -> Show Only Marked

- To view slide show 
1. Data Entry Mode
2. View -> Start Slide Show
